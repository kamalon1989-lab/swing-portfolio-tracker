Firebase Realtime Database를 연결하면 PC·스마트폰 어디서든
같은 포트폴리오 데이터를 사용할 수 있습니다.

📋 Firebase 설정 방법 (3분)
1. console.firebase.google.com → 프로젝트 추가
2. 좌측 메뉴 → 빌드 → Realtime Database → 데이터베이스 만들기
3. 위치: 아무거나 선택 → "테스트 모드에서 시작" 선택
4. 생성 후 표시되는 URL 복사 (예: https://my-project-xxx-default-rtdb.firebaseio.com)
5. 아래에 붙여넣기 → 저장
비워두면 클라우드 연동이 해제됩니다
