📊 TradingView 워치리스트:

AI에게 자기 포트폴리오를 정리해 달라고 할 때 아래 포맷으로 출력하라고 요청하면 됩니다. 그 후 "JSON 텍스트 붙여넣기"에 그대로 붙여넣고 적용하면 가져오기 완료.

📝 보유종목 포맷

{
  "type": "holdings",
  "version": 1,
  "items": [
    {
      "ticker": "AAPL",
      "name": "Apple Inc.",
      "shares": 10,
      "avgCost": 175.50,
      "buyDate": "2024-01-15",
      "lastBuyDate": "2024-03-20",
      "targetPrice": 220,
      "stopLoss": 160,
      "note": "장기 보유"
    }
  ]
}

필수: ticker, shares, avgCost · 나머지는 선택

📝 관심종목 포맷

{
  "type": "watchlist",
  "version": 1,
  "items": [
    {
      "ticker": "TSLA",
      "name": "Tesla",
      "targetBuy": 200,
      "note": "돌파 대기"
    }
  ]
}

필수: ticker · 나머지는 선택

💬 AI에게 보낼 프롬프트 예시

아래 내 포트폴리오를 다음 JSON 스키마로 정리해줘.
스키마: {"type":"holdings","version":1,"items":[{"ticker","name","shares","avgCost","buyDate","lastBuyDate","targetPrice","stopLoss","note"}]}
- ticker는 대문자 미국 거래소 티커
- buyDate/lastBuyDate는 YYYY-MM-DD
- 모르는 값은 필드를 빼거나 null
- 코드 블록 없이 JSON만 출력
